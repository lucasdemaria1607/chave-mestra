# TESOURO DOS ERROS

> "Tudo que não funcionou — documentado, lapidado, transformado em pergaminho."

Este diretório guarda o que deu errado. Cada fracasso documentado é um ativo.

## Como documentar um erro

1. **O que tentou:** descrição clara da ação
2. **O que esperava:** resultado pretendido
3. **O que aconteceu:** resultado real
4. **O que aprendeu:** insight extraído
5. **O que virou:** se virou pergaminho, roteiro, ou mudança de estratégia

## Regra

Nenhum erro é deletado. Todo erro é lapidado.
A jornada como produto. O fracasso como ativo.
