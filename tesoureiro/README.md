# Tesoureiro — Portal da Escala

Plugin 08 do Arsenal Chave Mestra. Diagnostica a fase do negócio (1–4), mapeia as alavancas corretas (Mídia/Código/Pessoas/Capital) e gera painel de métricas e próximo movimento estratégico.

## O que faz

Responde a pergunta que nenhum outro plugin toca: como crescer com o que já funciona? Enquanto os outros plugins respondem "o que criar" e "como vender", o Tesoureiro responde "o que otimizar", "onde reinvestir", "o que delegar" e "qual alavanca acionar agora".

## Quando usar

- "Diagnóstico do meu negócio"
- "Portal da Escala", "como crescer", "como escalar"
- "Análise de métricas", "ROAS", "LTV", "CAC"
- "Fase do negócio", "próximo movimento estratégico"
- "Onde reinvestir?", "o que delegar?"

## Skill

- `portal-da-escala` — 5 módulos: Diagnóstico de Fase, Mapa de Alavancas, Esteira de Valor, Painel de Métricas, Matriz de Decisão

## Output

Diagnóstico de Fase (1–4) + Alavanca Prioritária + Esteira de Valor com gaps + Painel de Métricas com semáforos + Próximo Movimento (1 principal + 2 suporte + 1 coisa para parar).

## Integração no Arsenal

```
Cartógrafo (01)         → [estado do mercado informa fase e alavanca]
Estrategista (06)       → [resultados de campanha alimentam métricas]
Arquiteto (07)          → [métricas registradas no Notion]
                       ↓
            TESOUREIRO (08)
                       ↓
Próximo Mapa de Campanha → [modalidade recomendada pela Matriz de Decisão]
```
