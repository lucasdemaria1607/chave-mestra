# Cartógrafo — Portal do Terreno

Plugin 01 do Arsenal Chave Mestra. Gera o Relatório de Terreno completo com 6 seções: mapa de players, panorama de ofertas, análise de canais, matriz de brechas, posicionamento estratégico e insumos para SZC.

## O que faz

Lê o que o mercado é — não o que sente. Antes de qualquer narrativa, copy ou campanha, existe um terreno com jogadores posicionados, ofertas que já prometem coisas e brechas que ninguém reivindicou. O Cartógrafo mapeia esse terreno com especificidade cirúrgica.

## Quando usar

- "Analisa o mercado de X"
- "Portal do Terreno", "relatório de terreno"
- "Quem são os players de X?", "brechas de posicionamento"
- Antes de criar produto, campanha ou posicionamento novo

## Skill

- `portal-do-terreno` — Relatório de Terreno com 6 seções

## Output

Relatório de Terreno: tabela de players + panorama de ofertas + análise por canal + lista de 3–6 brechas com evidência + 3 alternativas de posicionamento + insumos para SZC.

## Integração no Arsenal

```
[ENTRADA: nicho + produto + plataformas]
                       ↓
            CARTÓGRAFO (01)
                       ↓
Oráculo (02)           → [players informam o "inimigo" da persona]
Alquimista (03)        → [posicionamento vira espinha de C2]
Estrategista (06)      → [brechas de formato informam modalidade de lançamento]
```
