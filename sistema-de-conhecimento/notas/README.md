# Sistema de Conhecimento — Notas

> Esta pasta é monitorada pela Rotina de Accountability.
> Salve aqui suas notas em `.md` e faça commit. A rotina detecta pela data do último commit.

## Formato de nota

```markdown
# [Título da nota]

Data: YYYY-MM-DD
Tipo: insight / reflexão / observação / aprendizado / ideia
Fonte: [livro, conversa, filme, experiência, diário, etc.]

## A nota

[Conteúdo em suas palavras — não cópia, sua perspectiva]

## Conexões

- [Outra nota, ideia ou conceito que se conecta]

## Potencial de uso

- [ ] Conteúdo
- [ ] Estratégia
- [ ] Plugin / atualização do sistema
```

## Convenção de nomes

```
[YYYY-MM-DD]-[titulo-slug].md
Exemplo: 2026-04-28-paradoxo-da-originalidade.md
```

## Regra de ouro

Uma nota ruim capturada vale mais que um insight perfeito que ficou na cabeça.
