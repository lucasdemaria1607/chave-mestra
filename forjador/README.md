# Forjador — Forja do Conhecimento

Plugin 09 do Arsenal Chave Mestra. Captura e estrutura o aprendizado sistêmico via Zettelkasten digital, Tesouro dos Erros, evolução de plugins e mapa de versões do ecossistema.

## O que faz

O mecanismo de aprendizado do ecossistema. Não cria conteúdo para o público. Captura o que funcionou, o que falhou, e o que precisa evoluir — transformando cada erro em ativo e cada insight em atualização de sistema.

Lei central: um sistema que não aprende dos próprios erros está condenado a repeti-los.

## Quando usar

- "Documenta esse erro"
- "Registra esse aprendizado", "nota permanente"
- "Forja do Conhecimento", "Tesouro dos Erros"
- "O que aprendemos com essa campanha?"
- "Atualiza o plugin X com isso", "mapa de evolução"

## Skill

- `forja-do-conhecimento` — 4 módulos: Zettelkasten, Tesouro dos Erros, Evolução de Plugins, Mapa de Evolução

## Output

Nota Permanente / entrada no Tesouro dos Erros / Protocolo de Atualização de Plugin / entrada no Mapa de Evolução — conforme o contexto.

## Integração no Arsenal

```
[Qualquer campanha encerrada]
                       ↓
             FORJADOR (09)
                       ↓
Todos os plugins       → [recebem atualizações via Protocolo de Evolução]
Chaveiro (00)          → [coordena qual plugin atualizar e como]
```
