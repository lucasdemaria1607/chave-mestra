# GLOSSÁRIO — CHAVE MESTRA

---

## Termos Proprietários

| Termo | Definição |
|---|---|
| **Chave Mestra** | O ecossistema completo de criação e marketing para experts e produtores digitais. 7 camadas, 10 plugins, do estudo à escala. |
| **Chavideo** | Sistema de roteiros de vídeo virais — curtos (Reels, TikTok, Shorts) e longos (YouTube). Plugin 04. |
| **Chavossel** | Sistema de carrosséis editoriais — 10 slides = 10 atos narrativos. Plugin 05. Inclui plugin Figma com sintaxe proprietária. |
| **Chaveiro** | Plugin meta (00). Não produz conteúdo — mantém o sistema coerente. Recebe insumos brutos, extrai conhecimento, atualiza SKILL.md. |
| **Pergaminhos** | Qualquer conhecimento sistematizado dentro do universo Chave Mestra. |
| **Pergaminhos Brancos** | Conteúdo orgânico gratuito — topo de funil. Carrosséis, Reels, posts. |
| **Pergaminhos Pretos** | Materiais prontos para usar — low-ticket (R$27–R$97). Templates, roteiros, diagnósticos. |
| **Pergaminhos Dourados** | Conteúdo flagship — high-ticket. Liberação progressiva dentro da comunidade. |
| **Copy 3x5** | Framework estratégico: 3 Camadas (Emoção, Marca, Oferta) × 5 Níveis de Consciência. |
| **Método Carga** | Engenharia de densidade narrativa: Carga Intelectual, Emocional, Visual e de Sinalização. |
| **SZC** | Status Quo + Zeitgeist + Conclusão. Leitura do espírito do tempo de um nicho. |
| **Alma da Persona** | Mapa psicológico do lead: Inferno (de onde vem) + Purgatório (onde está preso) + Paraíso (para onde quer ir). |
| **Forja do Conhecimento** | Sistema de aprendizado permanente: Zettelkasten + Modelos Mentais + Método Científico + Progymnasmata. Plugin 09. |
| **A Ordem** | A comunidade dos que entraram no sistema. Compromisso com desenvolvimento, não clube. |
| **Portador** | Quem carrega as chaves — o lead/cliente que está na jornada. |
| **MINHA ESTEIRA** | Estrutura operacional no Notion — hub central de entregas, campanhas e gestão. |

## Universo Simbólico

| Expressão | Significado |
|---|---|
| "virar a chave" | Transformação, mudança de estado |
| "abrir o portal" | Acessar novo conhecimento ou nível |
| "ainda trancado" | Lead que ainda não chegou lá |
| "isso é tesouro" | Erro ou aprendizado de alto valor |
| "vira pergaminho" | Qualquer conhecimento sistematizado |
| "forjado na prática" | Conhecimento com experiência real |

## Símbolos Centrais

Chaves, portais, trancado/desbloqueado, baús, pergaminhos, selos, tesouros, mapas, forja, lapidação, maestria.

## Vocabulário de Mercado (não muda)

Funil, copy, lançamento, ROI, infoproduto, esteira, branding, lead, conversão, engajamento — são indexação cultural. Atraem. Permanecem.

## As 7 Camadas

| # | Nome | Função |
|---|---|---|
| C0 | Forja do Conhecimento | O que afila o expert permanentemente |
| C1 | Diagnóstico | O que mapeia o terreno antes de qualquer ação |
| C2 | Estratégia | O que decide o que criar, quando e com qual peso |

| C4 | Lançamento | Onde conteúdo vira campanha e receita |

| C6 | Escala | Onde o negócio cresce além do operacional |
