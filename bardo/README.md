# Bardo — Motor de Roteiros de Vídeo

Plugin 04 do Arsenal Chave Mestra. Gera roteiros de vídeo usando os 14 exercícios Progymnasmata e 20 técnicas cinematográficas de tensão.

## O que faz

Cria roteiros audiovisuais completos — não scripts genéricos, mas peças retóricas construídas sobre esqueleto Progymnasmata e calibradas com técnicas cinematográficas para maximizar retenção, engajamento, identificação ou viralização, dependendo do objetivo.

## Quando usar

- "Cria um roteiro de vídeo sobre X"
- "Chavideo", "script para Reels", "roteiro para YouTube", "TikTok"
- "Roteiro vertical" (60–90s) ou "roteiro horizontal" (2–5min)

## Skill

- `chavideo` — Roteiros com Progymnasmata + 20 técnicas cinematográficas

## Output

Roteiro beat a beat com marcação de tempo + nota técnica (exercício, mecânica principal, efeito retórico pretendido).
