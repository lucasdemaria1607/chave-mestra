# Chavideo — Motor de Roteiros via Progymnasmata + Tensão Cinematográfica

O Chavideo não gera scripts. Gera peças retóricas audiovisuais. Cada roteiro é um ensaio de vídeo construído sobre um esqueleto Progymnasmata e calibrado com técnicas cinematográficas de tensão para maximizar retenção, engajamento ou identificação — dependendo do objetivo.

**Lei de ouro:** a estrutura Progymnasmata dita o argumento. As técnicas cinematográficas ditam o ritmo. O Método Carga calibra a intensidade. Nenhuma das três opera sozinha.

---

## Anatomia do Roteiro Chavideo

Cada roteiro tem 5 camadas obrigatórias:

1. **Exercício Progymnasmata** — esqueleto argumentativo
2. **Combinação cinematográfica** — mecânica de tensão e retenção
3. **Calibração de Carga** — intensidade narrativa (ver Pergaminho de Copy)
4. **Formato** — vertical (60–90s), horizontal (2–5min), ou carrossel-vídeo (90–180s)
5. **CTA implícito ou explícito** — determinado pelo objetivo e nível de consciência do público

---

## Os 14 Exercícios Progymnasmata

### Seleção do Exercício

**Se o usuário fornecer o exercício:** usar diretamente.

**Se o usuário fornecer apenas tema/insumo:** diagnosticar pelo tipo de conteúdo:

| Diagnóstico | Exercício |
|---|---|
| História com moral → | Fábula |
| Relato de evento real/histórico → | Narrativa |
| Citação ou gesto de pessoa famosa → | Chria |
| Provérbio ou máxima → | Máxima/Gnome |
| Ataca mito ou crença popular → | Refutação |
| Defende posição com provas → | Confirmação |
| Amplifica vício ou virtude → | Lugar-Comum |
| Elogia pessoa, obra, lugar → | Elogio |
| Ataca ideia, vício, comportamento → | Vitupério |
| Compara dois elementos → | Comparação |
| Assume voz de personagem → | Etopeia |
| Descreve cena ou lugar vividamente → | Écfrase |
| Argumenta questão filosófica geral → | Tese |
| Discute utilidade/dano de norma ou lei → | Proposta de Lei |

## Os 20 Mecanismos Cinematográficos

### Grupo Intriga (abertura / manutenção de curiosidade)
1. **Information Gap** — abre lacuna irresistível nos primeiros 3s
2. **Open Loops** — mantém múltiplas promessas abertas durante o desenvolvimento
3. **Intrigue & Mystery** — clareza + omissão estratégica (você sabe mais do que entrega)
4. **Dramatic Question** — pergunta que sustenta toda a narrativa
5. **Foreshadowing** — planta expectativa do que vem

### Grupo Suspense (escalada e retenção)
6. **Cliffhanger** — corta em máxima tensão antes da revelação do clímax
7. **Rising Stakes** — problema → consequência irreversível → ação
8. **Withholding** — você sabe mais; decide quando revelar
9. **Contraste** — Antes/Depois, Verdade/Mentira, Grande/Pequeno

### Grupo Transformação (mudança de frame)
10. **Twist** — reorientação que invalida a premissa inicial
11. **Recontextualização** — mesmo fato; nova lente que muda tudo
12. **Narrative Misdirection** — conduz o público para conclusão errada — depois corrige
13. **Layered Revelation** — revela em camadas progressivas de profundidade

### Grupo Impacto (payoff e catarse)
14. **Catharsis** — libera a tensão acumulada com impacto emocional real
15. **Payoff** — entrega o que foi prometido na abertura — com peso proporcional
16. **Callback** — retoma imagem/frase da abertura com novo significado no final
17. **Time Compression** — comprime jornada longa em imagem densa (meses em segundos)

### Grupo Identidade (engajamento e viralização)
18. **Identity Threat** — ativa o ego questionando identidade ou crença do público
19. **Controlled Ambiguity** — deixa margem interpretativa — gera debate nos comentários
20. **Pattern Interrupt** — quebra expectativa para parar scroll nos primeiros 2s

## Estrutura do Roteiro

### Vertical (60–90s) — TikTok / Reels / Shorts

```
[00–03s] HOOK — Pattern Interrupt ou Information Gap
[03–10s] AMPLIFICAÇÃO — Rising Stakes ou Dramatic Question
[10–50s] CORPO — estrutura do exercício Progymnasmata (3–5 beats)
[50–70s] CLÍMAX — Twist ou Revelation
[70–90s] PAYOFF + CTA — Catharsis + próximo passo
```

### Horizontal (2–5min) — YouTube / Lives gravadas

```
[00–15s] HOOK + Dramatic Question
[15–45s] ESTABELECIMENTO — contextualização + Promise
[45–180s] CORPO — exercício Progymnasmata expandido com Open Loops
[180–240s] CLÍMAX — Layered Revelation ou Twist
[240–280s] PAYOFF + Callback
[280–300s] CTA explícito
```

## Processo de Criação

### Passo 1 — Diagnóstico do Insumo
- Identificar: essência, conflito central, argumento mais forte
- Definir: nível de consciência do público (N1–N5 do Pergaminho de Copy)
- Definir: objetivo (retenção / engajamento / identificação / viralização / autoridade)

### Passo 2 — Seleção do Exercício + Combinação
- Mapear o exercício Progymnasmata adequado
- Selecionar a combinação cinematográfica pelo objetivo
- Consultar `progymnasmata-video-map.md` para estrutura interna do exercício escolhido

### Passo 3 — Composição do Roteiro
- Escrever beat a beat seguindo a estrutura do exercício
- Aplicar os mecanismos cinematográficos nos pontos corretos
- Calibrar cargas narrativas (Intelectual / Emocional / Visual / Sinalização)

### Passo 4 — Revisão
- [ ] Os primeiros 3s param o scroll?
- [ ] Há pelo menos 1 loop aberto durante o corpo?
- [ ] O clímax justifica a atenção investida?
- [ ] O CTA tem um único próximo passo?
- [ ] Carga Visual está congruente com posicionamento?

## Regras

1. Nunca iniciar roteiro sem definir o exercício Progymnasmata
2. Nunca usar mecanismo cinematográfico sem identificar em qual beat opera
3. Os primeiros 3 segundos são inegociáveis — Pattern Interrupt ou Information Gap obrigatório
4. Não simplificar. O público Chave Mestra é inteligente e busca sofisticação
5. Tom: autoral, denso, provocativo — nunca linguagem de coach motivacional
