# O Iluminista — Plugin 10
## Tema Visual do Arsenal Chave Mestra

---

**Arquétipo:** O Iluminista — aquele que dá forma ao que os outros criam.

**Camada:** Transversal — aplica-se a todos os plugins e todas as camadas.

**Papel no sistema:**
O Iluminista não cria conteúdo. Ele veste o conteúdo com a forma que merece. Todo output do Arsenal — copy, roteiro, carrossel, campanha, relatório — pode ser transformado em artefato visual com a Estética Chave Mestra.

---

## Skills incluídas

| Skill | Arquivo | Descrição |
|-------|---------|-----------|
| Tema Visual | `skills/tema-visual/SKILL.md` | Sistema completo de design: 3 Pergaminhos, tokens, componentes, UX, formatos |

---

## Os 3 Pergaminhos

| Pergaminho | Estética | Usar para |
|------------|----------|-----------|
| **Branco** | Off-white, Cinzel, gold sutil | Educativo, técnico, autoridade, relatórios |
| **Preto** | Void black, ouro, glassmorphism | Provocação, lançamento, alta energia, impacto |
| **Dourado** | Pergaminho clássico, ornamentos | Transformação, premium, filosofia, legado |

---

## Formatos suportados

- HTML / CSS (interfaces, landing pages, dashboards)
- Instagram (carrossel 1080×1080, post, story)
- Slides / Apresentações (16:9, 4:3)
- PDF / Relatórios (A4, Letter)
- Documentos Word / DOCX
- PNG (posts estáticos, capas)

---

## Como instalar

1. Abra o projeto Chave Mestra no Claude Code
2. O plugin é ativado automaticamente via `.claude-plugin/plugin.json`
3. Para ativar manualmente: diga "aplica a estética Chave Mestra" ou "usa o Pergaminho Preto/Branco/Dourado"

---

## Conexão com outros plugins

O Iluminista é o último passo de qualquer pipeline:

```
[Qualquer Plugin] → conteúdo criado → O Iluminista → artefato visual pronto
```

- **Chavossel (05)** → veste o carrossel de 10 slides com design Instagram
- **Chavideo (04)** → formata o roteiro em template visual de script
- **Mapa de Campanha (06)** → cria o cronograma visual / dashboard de campanha
- **Portal da Escala (08)** → cria o painel de métricas com glassmorphism
- **Relatório de Terreno (01)** → formata em PDF/HTML estruturado com hierarquia visual
