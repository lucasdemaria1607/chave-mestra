# Alquimista — Pergaminho de Copy

Plugin 03 do Arsenal Chave Mestra. Calibra copy usando o sistema Copy 3×5 (C1/C2/C3) e o Método Carga (Intelectual/Emocional/Visual/Sinalização).

## O que faz

Diagnóstico de proporção por nível de consciência antes de escrever, e calibração de cargas narrativas após o rascunho. Integra dois frameworks: Copy 3×5 (arquitetura das camadas) e Método Carga (calibração de intensidade), garantindo que a copy opere abaixo da percepção racional.

## Quando usar

- "Escreve uma copy para X"
- "Calibra essa copy", "melhora esse texto"
- "Pergaminho de Copy", "Copy 3x5", "Método Carga"
- "Qual nível de consciência é esse público?"

## Skill

- `pergaminho-de-copy` — Copy 3×5 + Método Carga + 4 fases de resistência

## Output

Diagnóstico pré-escrita + copy calibrada + diagnóstico de cargas (Intelectual/Emocional/Visual/Sinalização) + checklist de resistência.
