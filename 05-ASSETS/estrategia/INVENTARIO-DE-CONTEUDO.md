# Inventário de Conteúdo — Conceitos do Sistema como Peças

Cada linha é um conceito real do sistema. Cada conceito é uma peça de conteúdo potencial.
A lógica: o sistema foi construído porque um problema existia. Esse problema é a entrada do conteúdo.
O conceito é o corpo. O reveal ("é por isso que existe no Chave Mestra") é o destino.

**Regra de uso:** não partir do conceito. Partir da dor/entrada. O conceito é a solução, não o assunto.
