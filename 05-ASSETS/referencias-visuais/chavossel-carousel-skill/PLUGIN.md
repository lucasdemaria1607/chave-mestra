# Chavossel Carousel Engine — Plugin Manifest

## Plugin Name
chavossel-carousel

## Description
Cria carrosseis completos no estilo Chavossel (Motor Oráculo 1.0) diretamente no Figma, com seleção inteligente de layouts, geração opcional de imagens via IA, e aplicação automática do design system gold/void de Lucas Grigo.

## Version
1.0.0

## Author
Lucas Grigo — Chave Mestra 2026

## Skills
- chavossel-carousel/SKILL.md — Core carousel creation engine

## Commands
- /create-carousel — Fluxo completo de criação de carrossel
- /generate-slide — Criar slide avulso
- /apply-chavossel — Aplicar design system em frames existentes
- /restyle — Trocar tipografia de um carrossel existente

## Required MCP Connections
- Figma (mcp__figma) — criação de frames, upload de imagens
- Nano Banana (mcp__nano-banana) — geração de imagens [opcional]
