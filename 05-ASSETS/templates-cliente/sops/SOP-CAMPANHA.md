# SOP — Campanha / Lançamento

**Tipo:** Complexo (sequenciado, alto impacto, muitas variáveis)
**Quando:** ao iniciar qualquer campanha, lançamento ou protocolo intensivo
**Tempo estimado:** 2–4 sessões de trabalho (dependendo da modalidade)
**Plugin responsável:** Alquimista (Forja de Oferta) + Arauto (Mapa de Campanha + Forja de Caixa)
**Pré-requisito:** §1 (DNA) + §2 (Persona + Terreno) do Dossiê preenchidos

---

## Pré-lançamento — Diagnóstico Anti-Impulso

Antes de qualquer planejamento, verificar os 5 sinais de risco:

| Sinal | Status | Ação se 🔴 |
|-------|--------|-----------|
| Audiência aquecida (≥30 dias de conteúdo recente) | [ ] 🟢  [ ] 🟡  [ ] 🔴 | Planejar fase de aquecimento antes de abrir vendas |
| Oferta validada (vendida ou testada 1x1) | [ ] 🟢  [ ] 🟡  [ ] 🔴 | Rodada de validação antes do lançamento em escala |
| Nível de consciência da audiência mapeado | [ ] 🟢  [ ] 🟡  [ ] 🔴 | Conteúdo educativo obrigatório no aquecimento |
| Histórico de entrega documentado | [ ] 🟢  [ ] 🟡  [ ] 🔴 | Reunir prova social mínima antes de abrir carrinho |
| Margem de tempo disponível (≥7 dias de aquecimento) | [ ] 🟢  [ ] 🟡  [ ] 🔴 | Reposicionar data de abertura |

**2+ sinais 🔴 = discutir com o cliente antes de prosseguir.**

## Fase 2 — Planejar a Campanha (Arauto — Mapa de Campanha)

### 2.1 Escolher modalidade

| Modalidade | Duração | Quando usar |
|-----------|---------|-------------|
| Meteórico | 3–7 dias | Audiência quente, urgência real, ticket baixo/médio |
| Desafio | 7/14/21 dias | Educar + converter, ticket médio/alto |
| Lançamento Interno | 4–8 semanas | Produto novo, audiência fria, construção de autoridade |
| Protocolo Luxo | 7–12 dias | Ticket alto, audiência qualificada, exclusividade |
| Protocolo Urgência | 48–72h | Cash rápido, lista aquecida, oferta conhecida |

### 2.2 Montar cronograma

- [ ] Ativar Arauto → Mapa de Campanha
- [ ] Definir: data de abertura do carrinho, data de fechamento
- [ ] Mapear 4 fases da campanha:

| Fase | Duração | Objetivo | Peças necessárias |
|------|---------|----------|-------------------|
| Aquecimento | [X dias] | Elevar nível de consciência | Roteiros/posts educativos |
| Abertura | [X dias] | Gerar desejo + urgência | Copy de lançamento, lives, emails |
| Conteúdo/Venda | [X dias] | Converter objeções | Depoimentos, perguntas frequentes, provas |
| Fechamento | [X dias] | Urgência final | Copy de escassez, bônus de última hora |

- [ ] Gerar cronograma dia-a-dia com copy por fase
- [ ] Salvar em `04-ESTRATEGIA/mapa-de-campanha.md`
- [ ] Atualizar §3 > Campanha do Dossiê

## Fase 3 — Produzir as Peças (Bardo + Alquimista + Iluminista)

- [ ] Listar todas as peças necessárias (do cronograma)
- [ ] Priorizar por urgência (primeiro o aquecimento, depois abertura, etc.)
- [ ] Para cada peça, acionar o plugin correto:

| Peça | Plugin |
|------|--------|
| Roteiro de vídeo de aquecimento | Bardo → Chavideo |
| Carrossel educativo | Bardo → Chavossel |
| Copy de e-mail de abertura | Alquimista → Pergaminho de Copy |
| Copy de post de fechamento | Alquimista → Pergaminho de Copy |
| Visual de oferta / card de bônus | Iluminista → Sistema de Design / Forja de Imagem |

- [ ] Salvar todas as peças em `05-CRIACAO/` por tipo
- [ ] Alocar no calendário: `06-OPERACAO/calendario-[mes].md`
- [ ] Registrar no §4 > Inventário de Outputs do Dossiê

## Fase 4 — Executar e Monitorar (Arauto — Esteira Notion)
