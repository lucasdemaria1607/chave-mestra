# SOP — Criação Semanal de Conteúdo

**Tipo:** Simples (ritmo, checklist)
**Quando:** toda semana, idealmente segunda ou terça
**Tempo estimado:** 90–120 min por semana
**Plugin responsável:** Bardo (Headline Generator + Chavideo + Chavossel) + Alquimista (Pergaminho de Copy)
**Pré-requisito:** §1 (DNA de Marca) e §2 (Persona) do Dossiê preenchidos

---

## Checklist semanal

### 1. Orientação (10 min)
- [ ] Ler Forja Semanal do `ritual-da-chave/SKILL.md` para o ciclo atual
- [ ] Confirmar tema/pilar da semana (calendário ou decisão da Lua Cheia anterior)
- [ ] Verificar no Dossiê: há alguma campanha ativa que o conteúdo deve sustentar? (§3 > Campanha)

### 2. Geração de headlines (20 min)
- [ ] Ativar Bardo → Headline Generator
- [ ] Informar: nicho, persona, tema, tom de voz (do §1 e §2 do Dossiê)
- [ ] Gerar banco de 20–30 headlines
- [ ] Selecionar 3–5 para produzir na semana
- [ ] Salvar banco em `05-CRIACAO/headlines/headlines-[YYYY-MM].md`

### 3. Produção de conteúdo (60 min)
Para cada headline selecionada, escolher o formato:

| Formato | Skill | Quando usar |
|---------|-------|------------|
| Vídeo curto (30–90s) | Chavideo → Script Creator | Alta energia, narrativa, demonstração |
| Carrossel (6–10 slides) | Chavossel | Ensino, conceito, lista, contraste |
| Copy de post | Pergaminho de Copy | Reflexão, provocação, bastidor |

- [ ] Criar roteiros/carrosséis/copy com o formato escolhido
- [ ] Salvar em `05-CRIACAO/roteiros/` ou `05-CRIACAO/carrosseis/` ou `05-CRIACAO/copy/posts/`
- [ ] Nomear conforme convenção: `[YYYY-MM-DD]-[tema-slug].md`

### 4. Revisão de copy (15 min)
- [ ] Passar cada peça pelo Copy Enhancer (Bardo) para ajustar para fala natural (roteiros)
- [ ] Verificar: a voz está alinhada com o DNA de Marca? (§1 do Dossiê)
- [ ] Verificar: a copy está calibrada para o nível de consciência da persona? (§2 do Dossiê)

### 5. Calendário (10 min)
- [ ] Alocar as peças no `06-OPERACAO/calendario-[YYYY-MM].md`
- [ ] Ou criar/atualizar entradas no Notion (Esteira Notion)
- [ ] Confirmar que há pelo menos 1 peça por dia útil da semana

### 6. Atualizar Dossiê
- [ ] Registrar no §4 > Inventário de Outputs as peças produzidas (tipo, qtd, skill usada)
