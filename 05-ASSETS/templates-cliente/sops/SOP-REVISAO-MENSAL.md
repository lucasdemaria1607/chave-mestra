# SOP — Revisão Mensal (Lua Cheia)

**Tipo:** Simples (ritmo mensal, checklist reflexivo)
**Quando:** fim de cada mês ou início do próximo
**Tempo estimado:** 60–90 min
**Plugin responsável:** Chaveiro (Ritual da Chave — Lua Cheia)

---

## Checklist de Revisão

### 1. Fechar o mês (15 min)
- [ ] Levantar métricas do mês: alcance, engajamento, cliques, vendas, novos seguidores
- [ ] Contar peças produzidas vs planejadas (% de execução)
- [ ] Identificar a peça com melhor performance e a com pior

### 2. Reavaliar as 6 dimensões (20 min)
- [ ] D1 Identidade — a voz está consistente? O universo de marca está sendo aplicado?
- [ ] D2 Inteligência — o público mudou? A persona ainda é válida?
- [ ] D3 Estratégia — o plano do mês foi cumprido? As metas foram atingidas?
- [ ] D4 Criação — o método está sendo usado? A qualidade subiu, desceu ou manteve?
- [ ] D5 Operação — o sistema está organizado? O calendário foi seguido?
- [ ] D6 Evolução — houve aprendizado documentado? Erros foram registrados?

> Atualizar semáforos no §0 do Dossiê se alguma dimensão mudou de status.

### 3. Registrar o que importa (15 min)
- [ ] 3 vitórias do mês (o que funcionou e por que)
- [ ] 3 aprendizados (o que não funcionou e o que vai mudar)
- [ ] 1 erro crítico → registrar no `09-EVOLUCAO/tesouro-dos-erros.md`
- [ ] Atualizar §6 do Dossiê

### 4. Planejar o próximo mês (20 min)
- [ ] Definir tema/foco do mês
- [ ] Verificar: há campanha planejada? (§3 do Dossiê)
- [ ] Revisar roadmap: ainda faz sentido? Alguma fase deve ser adiantada ou adiada?
- [ ] Confirmar calendário: `06-OPERACAO/calendario-[mes].md`

### 5. Decisão estratégica (10 min)
Preencher a matriz de decisão pós-ciclo no Dossiê (§6):

| Descoberta | Volta para | Ação decidida |
|-----------|------------|--------------|
| | | |

### 6. Fechar e versionar
- [ ] Gerar `09-EVOLUCAO/relatorio-[YYYY-MM].md` com o resumo do mês
- [ ] Atualizar HISTÓRICO DE VERSÕES no Dossiê
- [ ] Preparar Kit de Entrega (§7 do Dossiê) se houver fase encerrada neste mês
