# Oráculo — Forja da Persona

Plugin 02 do Arsenal Chave Mestra. Mapeia o avatar em 3 estados existenciais (Inferno/Purgatório/Céu) e gera a Matriz C1 3×5 de espelhamento.

## O que faz

Engenharia de espelhamento psicológico — não inventário de dores, mas dissecção cinematográfica dos 3 estados existenciais do avatar, com sensações físicas, linguagem nativa e objetivos secretos. O output alimenta diretamente C1 do Copy 3×5 e informa hook e fechamento de qualquer peça.

## Quando usar

- "Mapeia a persona do nicho X"
- "Forja da Persona", "Inferno Particular", "Alma da Persona"
- "Quem é meu público?", "descreve meu lead"
- Antes de criar copy, campanha ou conteúdo para novo nicho

## Skill

- `forja-da-persona` — 3 estados existenciais + Matriz C1 3×5

## Output

Inferno Particular + Purgatório Particular + Céu Particular (80–150 palavras cada, com sensações físicas) + Matriz C1 3×5 + Narrativa de Conexão.

## Integração no Arsenal

```
Cartógrafo (01)         → [players e oferta dominante informam o "inimigo" da persona]
                       ↓
              ORÁCULO (02)
                       ↓
Alquimista (03)        → [Inferno/Céu alimenta C1 diretamente]
Bardo (04)             → [Dramatic Question e Catharsis]
Escriba (05)           → [abertura e fechamento do carrossel]
Estrategista (06)      → [tom e narrativa da campanha]
```
