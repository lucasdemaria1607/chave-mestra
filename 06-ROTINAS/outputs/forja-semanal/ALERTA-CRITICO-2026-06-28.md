## O Fork — Duas Opções Reais

### Opção 1 — Redesenho Radical

O sistema atual tem atrito alto demais para o fluxo real de Lucas.

O protocolo ideal (Zettelkasten, notas permanentes, conexões emergentes) está funcionando como barreira em vez de andaime. A solução não é disciplina — é redesenho.

**O que isso significa na prática:**
- Abandonar o formato `.md` no repositório como ponto de entrada
- Capturar onde você já captura (voz, Notes, Telegram) e criar uma ponte periódica (quinzenal, mensal) para o sistema
- Reduzir o "sistema de conhecimento" a uma pasta simples e um commit por mês — sem protocolo, sem avaliação de qualidade, sem conexões obrigatórias
- Reformular o threshold de alerta: em vez de 7 dias, talvez 30 dias seja o ritmo real

**Gatilho para escolher esta opção:**
> Você quer manter o sistema vivo, mas o formato atual está errado.

---

### Opção 2 — Hibernação Formal

O sistema de conhecimento, como concebido, não é prioridade agora.

Não é falha. É honestidade sobre fases. Sistemas de conhecimento exigem energia cognitiva disponível — energia que pode estar, legitimamente, alocada em outras frentes (produto, campanha, operação).

**O que isso significa na prática:**
- Desativar a rotina de accountability (`rotina_accountability: pausada`)
- Manter o repositório sem obrigação de alimentação
- Definir uma data de retomada (ex: próximo trimestre, quando X campanha terminar)
- Documentar o estado atual no README para retomada fácil

**Gatilho para escolher esta opção:**
> O sistema é uma boa ideia, mas não para agora. Você quer voltar depois com cabeça livre.

## O Que Não Fazer

Continuar sem decidir é a terceira opção — e a pior.

Gera alertas toda semana que ninguém lê, mantém a sensação de "deveria estar fazendo" sem ação correspondente, e corrói a confiança no próprio sistema.

Se nenhuma das duas opções acima fizer sentido agora, a conversa correta é outra:
> *"O que está bloqueando uma decisão sobre o sistema de conhecimento?"*

Essa pergunta, respondida honestamente, provavelmente revela onde está o nó real.
