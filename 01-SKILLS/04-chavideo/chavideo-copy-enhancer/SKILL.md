# Chavideo Copy Enhancer

Transforms written scripts into spoken performance — cutting what doesn't land when said aloud, strengthening what does, creating rhythm that holds attention in motion.

**Principle:** A script that reads well on paper often dies on camera. A script that sounds natural when spoken usually looks strange on paper. Optimize for the ear, not the eye.

---

## When to Use

Trigger when:
- User says "o roteiro está engessado / formal / não flui"
- Script exists but sounds written, not spoken
- User wants to refine a generated script for delivery
- Structural markers leaked into the final text (violation to fix)
- Script is too long / too short for the target duration

---

## Core Distinctions: Written vs Spoken

| Written Copy | Spoken Copy |
|---|---|
| Complete sentences | Fragments allowed |
| Formal grammar | Conversational grammar |
| Reads visually | Sounds natural |
| Dense paragraphs | Short punchy units |
| Commas for flow | Pauses for performance |
| Verbs can be weak | Every verb must carry weight |
| Adjectives acceptable | Nouns and verbs do the work |

---

## Optimization Protocol

### Step 1 — Structural Audit (Remove Violations First)

Check for and eliminate:
- Any structural markers that leaked ("GANCHO:", "INTENSIFICADOR:", etc.)
- Bullet points in the final script
- Section headers within the narrative
- Any element that breaks the pure narrative flow

### Step 2 — Verbosity Elimination

For each sentence, apply:
- Remove filler phrases ("É importante notar que...", "Como podemos observar...")
- Cut redundant qualifiers ("muito", "bastante", "realmente" when adding nothing)
- Eliminate throat-clearing openings (sentences that delay the actual point)
- Compress: if an idea takes 30 words, find the 15-word version

**Before:** "Uma coisa que é muito importante que você entenda é que o problema que a maioria das pessoas enfrenta não é falta de conhecimento."
**After:** "O problema não é falta de conhecimento."

### Step 3 — Verb/Noun Strengthening

Replace weak verb + adjective with strong verb:
- "faz com que seja difícil" → "paralisa"
- "tem um efeito positivo em" → "transforma"
- "é uma coisa que acontece quando" → "surge quando"

Replace abstract nouns with concrete ones where possible:
- "o processo de desenvolvimento" → "a forja"
- "o resultado final" → "o que fica"

### Step 4 — Rhythm and Cadence

Create spoken rhythm through:
- **Sentence length variation:** Long. Short. Very short. Impact. Then medium again.
- **Repetition for emphasis:** "Não é sobre X. Nunca foi sobre X."
- **Pauses built into punctuation:** Em-dash creates pause. Period creates stop. Comma flows.
- **Tricolon (rule of three):** "Estuda. Testa. Refina."
- **Anaphora (repeated opening):** "Você que... Você que... Você que..."

### Step 5 — Cognitive Load Balancing

Each paragraph should have ONE main idea. Check:
- Is the lead processing one thing at a time?
- Are transitions clear enough for ears (not eyes)?
- Is the build logical when HEARD sequentially?

### Step 6 — Duration Calibration

Target benchmarks (spoken at natural video pace):
- 60s = ~150-180 words
- 90s = ~225-270 words
- 3min = ~450-540 words
- 10min = ~1500-1800 words

If too long: cut redundancy, not substance. Every cut should be an idea that doesn't earn its place.
If too short: deepen the Conteúdo Notável / desenvolvimento — not artificially pad.

```markdown
**Modificações principais:** [Lista de 3-5 principais ajustes feitos]

[SCRIPT OTIMIZADO COMPLETO — TEXTO CORRIDO SEM MARCAÇÕES]

---

## NOTAS DE ENTREGA (opcional)
*Dicas de performance para a câmera:*
- [Onde pausar para impacto]
- [Onde acelerar o ritmo]
- [Onde olhar diretamente para a câmera]
- [Onde o gesto ou expressão amplifica a palavra]
```

---

## Quality Standards After Enhancement

- ✅ Zero structural markers in text
- ✅ Reads naturally aloud in one pass without stumbling
- ✅ Each sentence earns its place
- ✅ Rhythm varies (no monotony)
- ✅ Duration matches target
- ✅ Dramatic Question still intact and clear
- ✅ Payoff still proportional to tension
- ✅ C1/C2/C3 integration preserved, not diluted by editing

---

## Common Enhancement Requests

**"Deixa mais pesado / mais impactante"**
→ Strengthen verbs, cut qualifiers, add tricolon, insert strategic pauses

**"Deixa mais leve / mais conversacional"**
→ Fragments, contractions, rhetorical questions, shorter sentences

**"Tá longo demais"**
→ Identify one idea per paragraph, cut any paragraph with two ideas, remove throat-clearing

**"Tá curto demais"**
→ Deepen the Conteúdo Notável — add specificity, data point, or story beat

**"Não tá fluindo"**
→ Read aloud, mark stumble points, restructure those sentences for spoken rhythm
