# Chaveiro — Meta-Plugin do Sistema

Plugin 00 do Arsenal Chave Mestra. Guardião do sistema — assimila novos conhecimentos, atualiza skills existentes, verifica coerência entre plugins e gera novos plugins quando necessário.

## O que faz

Garante que o sistema cresça sem se deteriorar. Princípio: crescimento por adição, nunca por substituição. Nenhum skill é sobrescrito sem prova de que a versão nova é superior. Todo conhecimento novo encontra seu lugar antes de ser integrado.

## Quando usar

- "Assimila esse livro" / "processa esse PDF"
- "Atualiza o skill X com isso"
- "Onde isso se encaixa no sistema?"
- "Confere se o sistema está coerente"
- "Gera um novo plugin para X"
- "Chaveiro", "integra esse conhecimento no sistema"

## Skill

- `chaveiro` — 5 protocolos: Assimilação de Livros, Atualização de Skill, Verificação de Coerência, Integração de Conhecimento Avulso, Geração de Novo Plugin

## Output

Varia por protocolo: Notas Permanentes + Cartões de Modelos Mentais + Propostas de Enriquecimento (Protocolo 1) / Análise de Delta + nova versão do skill (Protocolo 2) / Relatório de Saúde com semáforos (Protocolo 3).
