# Chaveiro — Plugin Meta de Manutenção v3.0

O guardião do sistema. Garante que o ecossistema Chave Mestra cresça sem se deteriorar.

## Skills

| Skill | Função |
|-------|--------|
| **Chaveiro** | Assimilação de livros/PDFs, atualização de skills, verificação de coerência entre plugins |
| **Forja do Conhecimento** | Zettelkasten digital, notas permanentes, modelos mentais, evolução de plugins |
| **Tesouro dos Erros** | Curadoria de erros críticos, extração de padrões, prevenção de falhas recorrentes |

## Posição no Pipeline

```
CAMADA META — opera sobre todas as outras camadas
Chaveiro ←→ Todos os plugins (manutenção + aprendizado + prevenção de erros)
```

## Princípios

1. **Nunca sobrescreva — sempre versione.** A versão anterior é patrimônio.
2. **Proponha antes de implementar.** O usuário decide; o Chaveiro executa.
3. **Erros são ativos.** Cada falha documentada previne a repetição.
4. **Glossário é lei.** Termos proprietários nunca são substituídos por genéricos.
