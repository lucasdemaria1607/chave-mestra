# Arauto — Plugin de Campanha e Operação v3.0

O mobilizador. Planeja campanhas, executa cronogramas e opera a máquina de lançamento.

## Skills

| Skill | Função |
|-------|--------|
| **Mapa de Campanha** | 4 modalidades de lançamento (Meteórico/Desafio/Interno/Forja de Caixa) + cronograma dia a dia + copy por fase |
| **Esteira Notion** | Integração com MINHA ESTEIRA: população de banco de dados, tarefas por dia, copy no corpo |
| **Forja de Caixa** | Campanhas intensivas: Luxo (posicionamento premium) e Urgência (caixa em 48-72h) |

## Posição no Pipeline

```
CAMADA 4 — LANÇAMENTO + OPERAÇÃO
Bardo (conteúdo pronto) → Arauto (campanha + Notion + execução) → Alquimista (métricas pós)
```

## Princípios

1. **Nunca planejar sem diagnosticar modalidade primeiro.**
2. **O arco narrativo dos 4 atos é inegociável.** Cada dia serve a um dos atos.
3. **Urgência só funciona quando genuína.** Nunca fake scarcity.
4. **Uma linha por dia no Notion.** Nunca agrupar dias.
