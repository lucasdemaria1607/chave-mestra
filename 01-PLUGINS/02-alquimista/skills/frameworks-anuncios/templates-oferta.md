# Templates de Oferta

10 ângulos narrativos para apresentar ofertas. Cada ângulo tem templates por tipo de produto — usando a taxonomia Chave Mestra como referência de implementação.

---

→ **Modo de operação:** os tipos de produto abaixo (Pergaminho, Portal, Forja, Mentoria Individual) são a taxonomia de referência CM. Para clientes externos, substituir pelos tipos de produto deles com seus nomes exatos. Ver `00-FUNDACAO/MODO-CLIENTE.md` para o protocolo de intake e tabela de equivalências.

**Tipos de produto CM (referência interna):**
- **Pergaminho** — material escrito (PDF, guia, método documentado)
- **Portal** — curso online (módulos, vídeo-aulas, acesso digital)
- **Forja** — intensivo presencial ou ao vivo (workshop, imersão, masterclass)
- **Mentoria Individual** — acompanhamento direto, alto ticket, relação 1:1

---

## Ângulo 1 — Autoridade / Educação

**Quando usar:** Público que ainda não conhece o produto. Constrói autoridade antes de vender.

### Pergaminho
```
Eu documentei [MECANISMO / MÉTODO] em [NÚMERO] páginas.

Não é um resumo. É o mapa completo — com os erros que custaram [CUSTO REAL] e os atalhos que nenhum [CONTEÚDO GRATUITO DO NICHO] vai te entregar.

[NOME DO PERGAMINHO] é para quem já sabe que precisa de [RESULTADO] e está cansado de colecionar conteúdo sem avançar.

→ [LINK / PALAVRA-CHAVE]
```

### Portal
```
Em [NÚMERO] módulos, eu destrinchei [TEMA] do início ao fim.

Sem enrolação. Sem conteúdo de redes sociais reformatado. Cada aula foi construída para transformar [SITUAÇÃO ANTES] em [SITUAÇÃO DEPOIS] — com método, não com motivação.

[NOME DO PORTAL] está aberto. Mas só para quem está pronto para [VERBO DE AÇÃO ESPECÍFICO].

→ [LINK / PALAVRA-CHAVE]
```

### Forja
```
Em [NÚMERO DE HORAS/DIAS], eu vou te levar de [PONTO A] até [PONTO B] — ao vivo, com método, com aplicação imediata.

Não é palestra. Não é teoria. É a [NOME DA FORJA] — um intensivo projetado para quem cansa de aprender e quer finalmente executar.

[DATA / FORMATO / ONDE]

→ [LINK / PALAVRA-CHAVE]
```

### Mentoria Individual
```
Em [NÚMERO] de encontros, eu trabalho direto com você em [PROBLEMA ESPECÍFICO].

Sem fórmula genérica. Sem conselho de palco. Eu analiso o que você tem, identifico o que está falhando e construo o caminho junto contigo.

[NOME DA MENTORIA] tem [NÚMERO] vagas. Não são abertas para todo mundo.

→ [LINK / PALAVRA-CHAVE]
```

## Ângulo 2 — Quebra de Objeção

**Quando usar:** Público com resistência ativa — preço, tempo, ceticismo, tentativas anteriores fracassadas.

### Pergaminho
```

[NOME DO PERGAMINHO] parte de onde os outros param. Não é mais uma lista de dicas. É [DESCRIÇÃO DO MECANISMO ÚNICO].

Se as outras leituras não funcionaram, é porque faltava isso.

→ [LINK / PALAVRA-CHAVE]
```

### Portal
```
"Já fiz [NÚMERO] cursos e não sai do lugar."

Cursos genéricos ensinam conceitos. O [NOME DO PORTAL] ensina [MECANISMO ESPECÍFICO] — a razão real pela qual [RESULTADO] funciona ou não.

A diferença está no método, não no volume de conteúdo.

→ [LINK / PALAVRA-CHAVE]
```

### Forja
```
"Não tenho tempo para [NÚMERO DE DIAS] de evento."

A [NOME DA FORJA] foi projetada exatamente para quem não tem tempo a perder.

Em [NÚMERO DE HORAS], você sai com [ENTREGÁVEL CONCRETO] pronto — não com anotações para aplicar depois.

Quem não tem tempo é quem mais precisa de precisão.

→ [LINK / PALAVRA-CHAVE]
```

### Mentoria Individual
```
"Já investi em mentoria e não tive retorno."

Mentorias genéricas entregam conselhos. A [NOME DA MENTORIA] entrega diagnóstico + execução.

Eu não te digo o que fazer. Eu trabalho com você enquanto você faz.

Retorno depende de precisão, não de intenção.

→ [LINK / PALAVRA-CHAVE]

## Ângulo 3 — Medo / Dor (Reflexão)

**Quando usar:** Público com dor ativa e urgente. Nomeia o problema com precisão antes de oferecer saída.

A diferença entre quem resolve e quem continua reclamando é uma decisão.

→ [LINK / PALAVRA-CHAVE]
```

### Portal
```
[SITUAÇÃO DE DOR] não é falta de esforço. É falta de método.

O [NOME DO PORTAL] existe para quem já está cansado de trabalhar duro sem ver o resultado proporcional ao esforço.

Você não está errado. Está usando a ferramenta errada.

→ [LINK / PALAVRA-CHAVE]
```

### Forja
```
Em [DATA], você vai poder olhar para [PROBLEMA ATUAL] e entender — com clareza cirúrgica — por que estava travado e o que fazer a partir de agora.

Ou pode continuar onde está.

A [NOME DA FORJA] é para quem escolhe o primeiro caminho.

→ [LINK / PALAVRA-CHAVE]
```

### Mentoria Individual
```
O problema não é que você não sabe. É que sem olhos de fora, você não consegue ver o que está faltando.

Eu já vi esse padrão. Já resolvi. E posso resolver com você.

[NOME DA MENTORIA] — [NÚMERO] vagas. Uma delas pode ser sua.

→ [LINK / PALAVRA-CHAVE]
```

## Ângulo 8 — Contraste / Comparação

**Quando usar:** Mercado com muitas alternativas. Posiciona o produto como a escolha certa ao mostrar o que o diferencia.

### Pergaminho
```
Você pode continuar coletando conteúdo gratuito sobre [TEMA].

Ou pode ter o mapa completo — estruturado, sequenciado e testado — em [NÚMERO] páginas.

[NOME DO PERGAMINHO] não é mais um conteúdo. É o sistema que organiza tudo o que você já sabe e preenche o que falta.

→ [LINK / PALAVRA-CHAVE]
```

### Portal
```
Cursos genéricos sobre [TEMA] ensinam o quê fazer.

O [NOME DO PORTAL] ensina por que funciona — e como adaptar para o seu contexto específico.

A diferença não é conteúdo. É profundidade de método.

→ [LINK / PALAVRA-CHAVE]
```

### Forja
```
Eventos motivacionais te deixam animado por 48 horas.

A [NOME DA FORJA] te deixa com [ENTREGÁVEL CONCRETO] — que você aplica na segunda-feira.

São formatos diferentes para objetivos diferentes. Escolha pelo objetivo.

→ [LINK / PALAVRA-CHAVE]
```

### Mentoria Individual
```
Consultoria entrega diagnóstico.
Curso entrega método.
Mentoria entrega as duas coisas — aplicadas ao seu caso.

A [NOME DA MENTORIA] não é consultoria. Não é curso. É o que acontece quando você tem um parceiro que conhece o terreno e trabalha com você dentro dele.

→ [LINK / PALAVRA-CHAVE]
```

## Ângulo 10 — Storytelling / Narrativa Pessoal

**Quando usar:** Construção de conexão com o público. Funciona especialmente bem para abertura de lançamentos e reativação de lista fria.

### Pergaminho
```
Eu passei [TEMPO / SITUAÇÃO] tentando resolver [PROBLEMA].

O que me tirou disso foi descobrir [MECANISMO] — que ninguém estava falando claramente.

Documentei tudo no [NOME DO PERGAMINHO] para que você não precise perder o que eu perdi.

→ [LINK / PALAVRA-CHAVE]
```

### Portal
```
Quando eu estava em [SITUAÇÃO ANTES], eu teria dado [VALOR / ALGO IMPORTANTE] para ter o que estou te entregando agora.

O [NOME DO PORTAL] é o que eu aprendi da forma difícil — reorganizado para que você aprenda da forma certa.

→ [LINK / PALAVRA-CHAVE]
```

### Forja
```
A [NOME DA FORJA] nasceu de um problema real que eu vi repetindo em [NÚMERO] de pessoas que me procuravam com [PROBLEMA RECORRENTE].

A solução não era mais conteúdo. Era imersão — com método, com acompanhamento, com entrega no final.

Foi assim que surgiu. É assim que funciona.

→ [DATA / LINK / PALAVRA-CHAVE]
```

### Mentoria Individual
```
Eu só aceito trabalhar com quem está comprometido de verdade.

Não porque sou seletivo por status. Porque mentoria que funciona exige as duas partes.

Se você chegou até aqui, provavelmente é o perfil certo.

[NOME DA MENTORIA] — [NÚMERO] vagas. Próximo passo: [PROCESSO DE APLICAÇÃO].

→ [LINK / PALAVRA-CHAVE]
```
