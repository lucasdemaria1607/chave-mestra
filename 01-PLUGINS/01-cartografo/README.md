# Cartógrafo — Plugin de Pesquisa e Diagnóstico v4.0

O explorador do terreno. Mapeia o mercado, disseca a persona e constrói o universo de marca antes de qualquer criação.

## Skills

| Skill | Função |
|-------|--------|
| **Portal do Terreno** | Análise estrutural de mercado: players, ofertas, canais, brechas, posicionamento |
| **Forja da Persona** | Mapeamento psicológico: Inferno/Purgatório/Céu + Matriz C1 3×5 + SZC + Alma da Persona |
| **Forja do Universo** | Worldbuilding + Primal Branding + StoryBrand: DNA simbólico da marca que alimenta C2 do Copy 3x5 |

## Posição no Pipeline

```
CAMADA 1 — DIAGNÓSTICO (tudo começa aqui)
Cartógrafo (terreno + persona + universo) → Alquimista (copy + oferta) → Bardo (conteúdo) → Arauto (campanha)

Portal do Terreno (mapa externo) + Forja da Persona (mapa humano) + Forja do Universo (mapa da marca)
= diagnóstico completo que alimenta C2 do Copy 3x5
```

## Princípios

1. **Estrutura antes de narrativa.** Mapeie o terreno antes de escrever uma palavra.
2. **Especificidade é obrigatória.** Nada genérico — nomes, números, exemplos concretos.
3. **A brecha é mais importante que o player.** O espaço vazio é a oportunidade.
4. **Mostre corpo.** Sensações físicas obrigatórias no mapeamento de persona.
5. **Quem constrói mundo governa percepção.** O universo de marca é a terceira perna do diagnóstico.
