# Bardo — Plugin de Conteúdo de Mídia v3.0

O criador. Transforma estratégia em conteúdo audiovisual e editorial de alto impacto.

## Skills

| Skill | Função |
|-------|--------|
| **Chavideo** | Roteiros de vídeo via 14 exercícios Progymnasmata + 20 mecanismos cinematográficos |
| **Chavossel** | Ensaios visuais (carrosséis 6-10 slides) via Progymnasmata + sintaxe v22 |
| **Headline Generator** | 30 headlines variadas e de alto impacto para vídeo curto |
| **Script Analyzer** | Engenharia reversa de scripts virais: hooks, gatilhos, mecânicas de retenção |
| **Copy Enhancer** | Otimização de copy para fala/delivery: cadência, verbos fortes, carga cognitiva |
| **Script Creator** | Criação completa de roteiros virais (60-90s) com estrutura validada |
| **Cavalo de Troia** | Arquitetura de conteúdo por imersão cultural — entra no mundo do lead via referência cultural/pop, transporta para o mundo da marca sem que ele perceba a travessia |

## Posição no Pipeline

```
CAMADA 3 — CRIAÇÃO
Alquimista (copy calibrada) → Bardo (vídeo + carrossel) → Arauto (campanha)
```

## Princípios

1. **Cada roteiro é uma peça retórica.** Progymnasmata dita o argumento, cinema dita o ritmo.
2. **Os primeiros 3 segundos são inegociáveis.** Pattern Interrupt ou Information Gap obrigatório.
3. **Não simplificar.** O público busca sofisticação. Nunca linguagem de coach motivacional.
4. **Lei do Ritmo.** Nunca três slides do mesmo tipo seguidos.
