# Chavideo Script Analyzer

Performs complete reverse engineering of viral video scripts — identifying the invisible architecture beneath the surface, extracting replicable patterns, and generating templates for other niches.

**Principle:** Every viral script that worked has invisible architecture. The job is to make it visible without killing it.

---

## When to Use

Trigger when:
- User sends a video script or transcript for analysis
- User asks "engenharia reversa de [video]"
- User wants to understand why a specific script worked
- User wants to extract patterns from successful content
- User sends a link/transcript of viral video from their niche

---

## Analysis Framework

### Layer 1 — Esqueleto Progymnasmata

Identify which classical exercise operates as the base:
- Which of the 14 exercises is the structural foundation?
- Are multiple exercises layered? (common in longer scripts)
- How does the exercise manifest in each section?

### Layer 2 — Técnicas Cinematográficas

Map all 20 techniques present:
- Which techniques appear in ABERTURA (0-3s)?
- Which sustain DESENVOLVIMENTO?
- Which close FECHAMENTO?
- How do they combine and reinforce each other?

### Layer 3 — Copy Total Integration

Analyze through C1/C2/C3 lens:
- **C1 (Emoção/Persona):** Which emotional mirror is used? Which pain/desire from Alma da Persona?
- **C2 (Marca/Tribo):** What symbolic language, tribal expressions, or worldbuilding?
- **C3 (Oferta/Racional):** What friction is removed? What logical pathway is created?

### Layer 4 — Gatilhos

Identify all 7 primary psychological triggers:
1. Recompensa, 2. Reconhecimento, 3. Objeção, 4. Medo, 5. Mistério, 6. Crença, 7. Popularidade

### Layer 5 — Dramatic Question

What is the central question that holds the entire script? The answer to this IS the script's architecture.

## Anti-Patterns to Flag

When analyzing, actively identify and flag:
- ❌ Twist aleatório (sem preparação prévia — falha técnica)
- ❌ Payoff desproporcional (prometeu e não entregou)
- ❌ Dramatic Question que some no meio (perde tensão)
- ❌ Retenção por confusão (não por mistério — diferença crítica)
- ❌ CTA desalinhado com conteúdo (quebra de coerência)
